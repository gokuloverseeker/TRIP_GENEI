# TripGenie AI — AMD Edition

> TripGenie AI is a full‑stack AI-powered travel optimization engine that generates personalized, budget‑optimized itineraries in under 30 seconds. Built with AMD ROCm acceleration and a resilient triple‑fallback AI architecture, it delivers smart hotel recommendations, dynamic budget allocation, and real-time CO₂ footprint tracking through a modern interactive dashboard.
